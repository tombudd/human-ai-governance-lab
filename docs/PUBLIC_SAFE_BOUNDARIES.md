# Public-Safe Boundaries

This repository is intentionally clean-room.

## Included

- Toy task examples
- Deterministic Python rules
- Unit tests
- Sample receipts
- Public-safe documentation

## Excluded

- Private logs
- Internal schemas
- Real system prompts
- Proprietary architecture
- Secrets or credentials
- Claims of autonomous intelligence
- Claims of production readiness

## Positioning Rule

The project is about the public problem of reviewable, testable, accountable AI-assisted work. It is not a disclosure vehicle for private systems.
